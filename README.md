# 🚀 Premium Sub4Unlock Smart Link Locker Tool Script

A highly optimized, serverless, and 100% mobile-responsive **Sub4Unlock** link shortening and locking tool script. Built purely with HTML5, CSS3, and Vanilla JavaScript. It uses the **URL Hash (#) Method** to guarantee 100% stability, completely bypassing Blogger's country-specific redirects and mobile URL bugs (`?m=1`).

---

## ✨ Features
* 📱 **Modern Pill UI:** Beautifully crafted layout matching premium platforms like `sub4unlock.io`.
* ⚡ **100% Serverless:** No databases, no login requirements, and no hosting fees. Everything is processed client-side.
* 🔒 **Smart Hash Encryption:** Encodes destination data securely inside URL hashes (`#`) making it bypass-proof.
* 🚦 **Multi-Platform Support:** * YouTube (Subscribe, 2nd Subscribe, Like & Subscribe, Sub & Hit Bell, Like Video, Like & Comment)
  * Instagram (Follow Profile)
  * Facebook (Follow Page)
  * Telegram (Join Group/Channel)
* ⏱️ **Unlock Guard Timer:** Built-in 4-second delay mechanism preventing fake clicks.

## 🚀 How to Use on Blogger

1. Log in to your **Blogger Dashboard**.
2. Create a **New Page** (Recommended) or a New Post.
3. Switch the editor from Visual to **HTML View**.
4. Copy the complete source code from `index.html` in this repository.
5. Paste it into the Blogger HTML view and click **Publish**!

## 🌐 Live Demo Deployment (GitHub Pages)

You can easily host this tool as a standalone live website using GitHub Pages:
1. Go to your repository's **Settings** tab.
2. On the left sidebar, click on **Pages**.
3. Under **Build and deployment**, set the Source branch to `main` (or `master`) and folder to `/ (root)`.
4. Click **Save**. Within a few minutes, your tool will be live at `https://your-username.github.io/sub4unlock-blogger-script/`
